Thoughts API

Welcome to the thoughts API!

Endpoints:

  • /thoughts GET or POST to get all the thoughts or create a new thought. Use this if you're only interested in thoughts and not comments.
  • /thoughts_comments GET to get all the thoughts as well as the number of associated comments.
  • /thoughts/:ThoughtID GET a specific thought from it's ThoughtID.
  • /comments GET or POST to get all the comments or create a new comment.
  • /comments/:ThoughtID GET all comments for a specific thought.
